# jquery-globals

jquery helper library to using in esm module 

# install 

```
npm install jquery @easylogic/jquery-globals
```


# not woking in webpack 

```js
import $ from 'jquery'
window.jQuery = $ 
import 'summernote/dist/lang/summernote-ko-KR'; 
```

# Use jquery-globals  

```js
import 'jquery-globals'
import 'summernote/dist/lang/summernote-ko-KR'; 
```
